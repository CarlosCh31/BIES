# Proyecto Planeta BIES

Este proyecto se basa en el planeta BIES y todos los animales que viven en este.
Autores:
Carlos Chavarria
Amanda Esquivel

## Requisitos

- JDK 21
- Maven 3.8.1

## Instrucciones para Compilar

mvn clean install
mvn exec:java -Dexec.mainClass="una.ac.Main"
